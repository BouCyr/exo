# Multiplication Game (Exo)

A simple, mobile-first multiplication game for children with a neon aesthetic.

## Features
- **PWA Ready**: Can be installed on your phone or desktop.
- **Neon Aesthetic**: Fun colors that cycle as you progress.
- **Score Tracking**: Encouraging messages in French.
- **Customizable**: Change `TABLE_LIMIT` in `index.html` to adjust difficulty.

## Deployment to GitHub Pages
1. Go to your repository on GitHub.
2. Click on **Settings** > **Pages**.
3. Under **Build and deployment**, select `Deploy from a branch`.
4. Select `main` (or your primary branch) and the `/ (root)` folder.
5. Click **Save**.
6. Your site will be live at `https://<your-username>.github.io/<repo-name>/`.
